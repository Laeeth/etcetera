class Stack(T);

A generic last-in-first-out (LIFO) stack implementation.

Examples

auto stack = new Stack!(int);

assert(stack.empty);
assert(stack.count == 0);

int limit = 1_000_000;

for (int x = 1; x <= limit ; x++)
{
	stack.push(x);
}

assert(stack.peek() == limit);
assert(stack.count == limit);
assert(stack.contains(1));
assert(stack.contains(limit));
assert(!stack.empty);

for (int x = limit; x >= 1 ; x--)
{
	assert(stack.pop() == x);
}

assert(stack.empty);

for (int x = 1; x <= limit ; x++)
{
	stack.push(x);
}

stack.clear();

assert(stack.empty);
assert(stack.count == 0);
assert(!stack.contains(1));
assert(!stack.contains(limit));

this(size_t minSize = 32000);

Construct a new stack.

By default the stack is allocated 32k bytes. Once the stack grows above that limit it is rellocated to use double, ad infinitum. If the stack reduces to use only half of its allocation the allocation is halfed. The stack will never uses below the minimum allocation amount.

Parameters

size_t minSize The minimum size of the stack. Set to 32kb by default.

void push(T item);

Push an item onto the stack.

Parameters

T item The item to push onto the stack.

T peek();

Return the latest item pushed onto the stack.

Return Value

The last item pushed onto the stack.

Exceptions Thrown

AssertError if the stack is empty.

T pop();

Return the latest item pushed onto the stack and remove it from the stack.

Return Value

The last item pushed onto the stack.

Exceptions Thrown

AssertError if the stack is empty.

size_t count();

Get the number of items stored in the stack.

Return Value

The number of items stored in the stack.

bool empty();

Test if the stack is empty or not.

Return Value

true if the stack is empty, false if not.

bool contains(T item);

Check if a value is contained in the stack.

Parameters

T item The item to find in the stack.

Return Value

true if the item is found on the stack, false if not.

void clear();

Clear the stack.
